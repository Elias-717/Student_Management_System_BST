# Student Record Management System (BST)

Java console app for managing student records using a Binary Search Tree (BST). Includes insert/search/delete, traversals, utilities, and sorting demos.

## Requirements
- Java 8+ (JDK)

## How to run
- Compile: `javac Main.java`
- Run: `java Main`

## Files
- `Main.java` — demo runner and test cases
- `BST.java` — BST implementation and traversals
- `Node.java`, `Student.java` — data structures
- `Sorting.java` — merge sort (name) and quick sort (CGPA)
- `students.csv` — sample data

## Notes
- The BST rejects duplicate matric numbers.
- Traversal helpers: `inOrderList()`, `preOrderList()`, `postOrderList()`.
